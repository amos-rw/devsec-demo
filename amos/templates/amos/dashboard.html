About this service

The User Authentication Service (UAS) is a production-style Django application that covers the complete authentication lifecycle — registration, login, session management, password changes, and secure sign-out.

It is built on Django's battle-tested built-in auth system and follows security best practices: CSRF protection on every form, open-redirect guards on login redirects, session hash rotation on password change, and POST-only logout to prevent CSRF-based sign-out attacks.

All passwords are hashed with PBKDF2-SHA256 and validated through Django's password-strength pipeline before they are accepted.
